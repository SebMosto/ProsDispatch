- **Severity:** P0 — full app failure post-login
- **Steps to reproduce:** Login → redirect to /dashboard succeeds → navigate to /clients or hard-reload → infinite spinner

- **Root cause:** VITE_SUPABASE_URL and/or VITE_SUPABASE_ANON_KEY missing from Vercel production env vars. Vite bakes these at build time. Missing vars cause Supabase client to hang silently on getSession().
